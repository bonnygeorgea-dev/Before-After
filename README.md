# Before / After Maker (Batch) — EMD-KOR

Single-file, offline HTML tool. Add **multiple Before/After pairs** at once, then
share them all to WhatsApp or download them — each pair as its own image, with
labels and an **EMD - KOR** watermark.

- No backend, no upload — everything runs on the device.
- Add / reorder / delete pairs; swap before↔after per pair.
- Global settings (layout, fill/fit, labels, watermark, date) apply to every pair.
- **WhatsApp all** — one share sheet sends every pair as a separate photo.
- **Download all** — saves each pair as a separate PNG.
- Mobile-friendly with a sticky export bar; camera or gallery on phones.

## Use
Open `index.html` in Chrome, or visit the GitHub Pages link once deployed.

## Files
- `index.html` — the app
- `.nojekyll` — tells GitHub Pages to serve files as-is
